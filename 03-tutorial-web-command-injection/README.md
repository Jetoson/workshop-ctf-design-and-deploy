# Web Command Injection

**Web Command Injection** is a web security CTF challenge.
The goal is to exploit a command injection vulnerability in a PHP application to retrieve the flag.

## Requirements

* **Docker**

## Challenge Structure

* **`src/`**: PHP source code.
* **`deploy/`**: Deployment configuration (Dockerfile).
* **`solve/`**: Solution script and Docker environment.

## The Flag

The flag is located at `/flag` inside the container.

## Getting Started

1. Go to `deploy/` to run the challenge.
1. Go to `solve/` to run the exploit.
